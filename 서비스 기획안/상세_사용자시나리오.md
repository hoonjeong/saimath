# 사이수학 입학테스트 - 사용자 시나리오

## 시나리오 1: 관리자 - 신규 시험지 만들기

**등장인물**: 김선생님 (학원 관리자)  
**목표**: 초등 3학년 1학기 입학테스트용 시험지 만들기  
**소요 시간**: 약 40분

### 상황
새 학기가 시작되어 입학 상담이 많아졌다. 현재 초3 학생들의 실력을 정확히 파악하기 위해 새로운 입학테스트를 준비해야 한다. 최근 교육과정에 맞춰 문제를 새로 구성하려고 한다.

### 단계별 행동

#### 1단계: 시작 (9:00 AM)
- 컴퓨터를 켜고 관리자 페이지에 로그인
- 대시보드에서 "📝 시험지 만들기" 카드 클릭
- **생각**: "25문제면 충분하겠지. 연산, 도형, 문장제를 골고루 넣어야겠어"

#### 2단계: 기본 설정 (9:02 AM)
- 학년 선택 화면에서 "초3" 버튼 클릭
- 학기 선택에서 "1학기" 버튼 클릭
- [다음] 버튼 클릭
- **생각**: "1학기는 분수랑 나눗셈이 중심이니까 그쪽 비중을 좀 높여야지"

#### 3단계: 1~5번 문제 입력 - 연산 기본 (9:03~9:15 AM)
- 1번 문제: 두 자리 수 덧셈
  - 이미지 없이 선지만 입력
  - ① 34, ② 44, ③ 54, ④ 64
  - 정답: ③ 선택
  - [다음 문제] 클릭

- 2번 문제: 세 자리 수 뺄셈
  - 문제 이미지 업로드 (계산 과정이 있는 문제)
  - 이미지 파일 선택 → 미리보기 확인
  - 선지 4개 입력
  - 정답 선택
  - [다음 문제] 클릭

- 3~5번: 비슷한 방식으로 연산 문제 입력
- **생각**: "이미지 업로드가 생각보다 빠르네. 미리보기로 확인하니까 실수가 없어"

#### 4단계: 6~10번 문제 입력 - 분수 (9:15~9:25 AM)
- 6번 문제: 분수 크기 비교
  - 이미지 업로드 (분수를 그림으로 표현)
  - 선지 입력
  - 정답 선택
  - [다음 문제] 클릭

- 아차! 정답 선택을 깜빡함
- 에러 메시지: "정답을 선택해주세요" (빨간색)
- 정답 선택 후 다시 [다음 문제]
- **생각**: "다행히 넘어가기 전에 체크해주네. 실수 방지에 좋아"

- 7~10번: 분수 계산, 분수와 소수 관계 문제 입력

#### 5단계: 잠깐 휴식 & 임시 저장 (9:25 AM)
- 전화벨이 울림 → 상담 전화
- 통화하면서 화면 위 [임시 저장] 버튼 클릭
- "임시 저장되었습니다" 토스트 메시지
- 10분간 전화 상담
- **생각**: "임시 저장 기능 정말 유용해. 데이터 날아갈까봐 걱정했는데"

#### 6단계: 재개 (9:35 AM)
- 전화 끝나고 페이지 새로고침
- 10번 문제까지 자동으로 저장되어 있음
- 11번 문제부터 이어서 작성

#### 7단계: 11~20번 문제 입력 - 도형, 측정 (9:35~9:50 AM)
- 11번: 삼각형 넓이 구하기
  - 도형 이미지 업로드
  - 계산 선지 입력
  
- 15번: 길이 단위 변환
  - 이미지 없이 텍스트 문제
  
- 진행 상황 확인: "20 / 25"
- **생각**: "벌써 80% 완성이네. 조금만 더!"

#### 8단계: 21~25번 문제 입력 - 문장제 (9:50~10:00 AM)
- 21번: 사탕 나눠주기 문제
  - 긴 문장이라 이미지로 업로드
  - 선지 입력
  
- 25번: 시계 보기 문제
  - 시계 이미지 업로드
  - 선지: ① 3시 15분, ② 3시 30분, ③ 3시 45분, ④ 4시
  - 정답: ③ 선택

- 진행 상황: "25 / 25" ✓
- **생각**: "드디어 끝! 난이도 배분도 적절한 것 같아"

#### 9단계: 최종 확인 및 저장 (10:00 AM)
- [저장하기] 버튼 클릭 (파란색으로 활성화)
- 확인 팝업: "시험지를 저장하시겠습니까?"
- [확인] 클릭
- "저장 중..." 로딩 표시 (3초)
- "저장되었습니다!" 성공 메시지
- 자동으로 관리자 홈으로 이동
- **생각**: "완료! 이제 학생들 테스트 보게 하면 되겠다"

### 결과
- ✅ 초3-1학기 시험지 25문제 완성
- ✅ 연산 10문제 + 분수 5문제 + 도형/측정 5문제 + 문장제 5문제
- ✅ 이미지 15개, 텍스트만 10개
- ✅ 약 1시간 소요 (전화 상담 포함)

## 시나리오 2: 학생 - 첫 입학테스트 응시

**등장인물**: 박민준 (초등 3학년 예비 학생)  
**목표**: 사이수학 입학테스트 완료  
**소요 시간**: 약 30분

### 상황
민준이는 엄마와 함께 학원 상담을 받으러 왔다. 선생님이 "온라인으로 간단히 테스트를 보고 결과를 바로 확인할 수 있어요"라며 태블릿을 건네주었다. 민준이는 수학을 좋아하지만 시험은 좀 긴장된다.

### 단계별 행동

#### 1단계: 시작 화면 (2:00 PM)
- 태블릿 화면에 "사이수학 입학테스트" 표시
- 안내 문구 읽기:
  - "총 25문제가 출제됩니다"
  - "각 문제당 4점씩 배점됩니다"
- [시작하기] 버튼 클릭
- **심리**: "25문제면 많지 않네. 할 수 있어!"

#### 2단계: 정보 입력 (2:01 PM)
- 이름 입력란에 "박민준" 입력
- 전화번호 입력란 클릭 → 엄마께 물어봄
- 엄마가 알려준 번호 입력: "01012345678"
- 자동으로 "010-1234-5678"로 하이픈 생김
- **생각**: "오, 자동으로 바뀌네"

- 학년 선택 드롭다운 클릭 → "초등 3학년" 선택
- 학기 선택: "1학기" 라디오 버튼 클릭
- [시작하기] 버튼 클릭
- **심리**: "드디어 시작이다. 긴장된다..."

#### 3단계: 로딩 (2:02 PM)
- "시험지 불러오는 중..." 로딩 화면 (2초)
- 프로그레스 바가 빙글빙글
- **심리**: "빨리 시작하고 싶어"

#### 4단계: 1번 문제 (2:02 PM)
- 화면 상단: "1 / 25" 표시
- 문제: "23 + 31 = ?"
- 선지 4개가 큰 카드로 표시:
  - ① 34
  - ② 44
  - ③ 54
  - ④ 64
  
- ③ 54 카드를 손가락으로 터치
- 카드 배경이 파란색으로 변함 (선택 표시)
- 하단 [다음] 버튼이 활성화 (파란색으로)
- [다음] 버튼 클릭
- **심리**: "쉽네! 다음 문제도 잘 풀 수 있을 것 같아"

#### 5단계: 2~10번 문제 (2:02~2:12 PM)
- 2번: 세 자리 수 뺄셈 (이미지 문제)
  - 이미지를 손가락으로 확대해서 자세히 봄
  - 계산 후 답 선택 → [다음]
  
- 5번: 곱셈 문제
  - 암산으로 계산
  - 답 선택 → [다음]
  
- 8번: 분수 크기 비교
  - 조금 고민됨 (15초 정도)
  - 그림을 보고 답 선택 → [다음]
  - **생각**: "분수는 좀 어려운데... 맞았으면 좋겠다"

- 진행 상황 확인: "10 / 25" (프로그레스 바 40%)
- **심리**: "벌써 10개! 반 정도 왔어"

#### 6단계: 11~20번 문제 (2:12~2:22 PM)
- 13번: 삼각형 넓이 (이미지 문제)
  - 공식 기억해내기... "밑변 × 높이 ÷ 2"
  - 계산 후 답 선택
  - **생각**: "이거 배웠는데... 맞게 했나?"

- 17번: 길이 단위 변환
  - "1km = 1000m" 기억해냄
  - 답 선택
  - **생각**: "이건 자신있어!"

- 진행 상황: "20 / 25" (80%)
- **심리**: "이제 5개만 더! 파이팅!"

#### 7단계: 21~24번 문제 (2:22~2:28 PM)
- 21번: 사탕 나눠주기 문제 (긴 문장)
  - 문제를 천천히 두 번 읽음
  - 선지를 보고 계산
  - 답 선택
  - **생각**: "문장제는 잘 읽어야 해"

- 23번: 시계 보기
  - 시계 이미지 보기
  - "3시 반 조금 전이니까... 3시 15분?"
  - ① 선택 → [다음]

#### 8단계: 25번 문제 - 마지막 (2:28 PM)
- 화면 상단: "25 / 25" (프로그레스 바 100%)
- 문제: 복잡한 계산 문제
- 신중하게 계산 (30초)
- 답 선택
- [제출하기] 버튼 표시 (이전과 다르게 초록색)
- **심리**: "드디어 마지막이다. 제출하면 끝!"

- [제출하기] 클릭
- 확인 팝업: "제출하시겠습니까? 제출 후에는 수정할 수 없습니다."
- **심리**: "진짜 제출해도 되나? (3초 고민) 그래, 하자!"
- [확인] 클릭

#### 9단계: 채점 중 (2:29 PM)
- "채점 중..." 로딩 화면 (3초)
- 빙글빙글 돌아가는 애니메이션
- **심리**: "떨린다... 몇 점 나왔을까?"

#### 10단계: 결과 확인 (2:29 PM)
- 화면 중앙에 큰 숫자: "84점"
- 하단에 상세 정보:
  - ✓ 21개 정답
  - ✗ 4개 오답
  - 정답률: 84%
- 메시지: "👍 잘했습니다! 좋은 결과예요!"
- 안내: "결과는 관리자가 확인할 수 있습니다"

- **심리**: "84점! 생각보다 잘 봤다! 엄마한테 보여줘야지!"
- 엄마께 태블릿 보여주며 "엄마, 나 84점 나왔어!"

#### 11단계: 종료 (2:30 PM)
- 선생님이 결과 확인
- "잘했어요, 민준이! 연산 실력이 좋네요. 분수는 조금 더 연습하면 좋겠어요"
- 민준이: "네! 열심히 할게요!"
- **심리**: "학원 다니면 더 잘할 수 있을 것 같아!"

### 결과
- ✅ 25문제 전부 풀이 완료
- ✅ 84점 (21개 정답)
- ✅ 약 28분 소요
- ✅ 학생과 관리자 모두 즉시 결과 확인
- ✅ 긍정적인 경험으로 입학 의향 상승

## 시나리오 3: 관리자 - 학생 결과 분석 및 상담

**등장인물**: 김선생님 (학원 관리자)  
**목표**: 민준이의 테스트 결과 분석 후 학부모 상담  
**소요 시간**: 약 15분

### 상황
민준이가 테스트를 마치고 30분 후, 학부모와 정식 상담 시간이 되었다. 민준이의 결과를 자세히 분석해서 강점과 약점을 파악하고, 맞춤 학습 계획을 제안해야 한다.

### 단계별 행동

#### 1단계: 결과 페이지 접속 (3:00 PM)
- 관리자 페이지 로그인
- "📊 결과 보기" 메뉴 클릭
- 학생 목록 화면 표시
- **생각**: "민준이 결과를 자세히 봐야겠어"

#### 2단계: 학생 검색 (3:01 PM)
- 상단 검색창에 "박민준" 입력
- 검색 결과 1건 표시:
  ```
  | 이름   | 학년 | 학기  | 응시횟수 | 최근응시일 | 최고점수 |
  | 박민준 | 초3  | 1학기 | 1회     | 2025.11.13 | 84점    |
  ```
- 행 클릭 → 학생 상세 페이지로 이동
- **생각**: "첫 응시네. 84점이면 나쁘지 않은데"

#### 3단계: 응시 이력 확인 (3:02 PM)
- 학생 정보 카드 확인:
  - 이름: 박민준
  - 전화번호: 010-1234-5678
  - 총 응시 횟수: 1회
  - 평균 점수: 84점

- 응시 이력 테이블:
  ```
  | 응시일        | 학년/학기  | 점수 | 정답수 | 상세보기 |
  | 2025.11.13 14:29 | 초3-1학기 | 84점 | 21개  | [보기]  |
  ```

- [보기] 버튼 클릭 → 상세 결과 화면으로
- **생각**: "어떤 문제를 틀렸는지 확인해봐야겠어"

#### 4단계: 문제별 결과 분석 (3:03~3:10 PM)
- 상세 결과 화면 스크롤하며 확인

**연산 부분 (1~10번)**
- 1~7번: 모두 정답 ✓ (파란색)
- 8번: 분수 크기 비교 ✗ (빨간색)
  - 학생 답안: ②번
  - 정답: ③번
- 9~10번: 정답 ✓

- **메모 작성**: "분수 크기 비교 약함"

**도형/측정 부분 (11~20번)**
- 11~13번: 정답 ✓
- 14번: 넓이 단위 환산 ✗
  - 학생 답안: ①번
  - 정답: ④번
- 15~17번: 정답 ✓
- 18번: 각도 측정 ✗
  - 학생 답안: ③번
  - 정답: ②번
- 19~20번: 정답 ✓

- **메모 작성**: "단위 환산, 각도 개념 보완 필요"

**문장제 부분 (21~25번)**
- 21~24번: 정답 ✓
- 25번: 복잡한 계산 ✗
  - 학생 답안: ②번
  - 정답: ④번

- **메모 작성**: "문장제 이해력 좋음, 복잡한 계산 실수"

#### 5단계: 종합 분석 (3:10 PM)
**강점**:
- ✅ 기본 연산 (덧셈, 뺄셈, 곱셈) 매우 우수
- ✅ 문장제 이해력 좋음
- ✅ 도형 기본 개념 이해

**약점**:
- ⚠️ 분수 개념 (크기 비교)
- ⚠️ 단위 환산 (넓이, 각도)
- ⚠️ 복잡한 다단계 계산

**추천 학습 계획**:
1. 분수 집중 학습 (2주)
2. 측정 단원 복습 (1주)
3. 계산 정확도 향상 훈련

- **생각**: "분수를 확실히 잡아주고, 측정 쪽 보완하면 90점 이상 충분히 가능하겠어"

#### 6단계: 학부모 상담 (3:12~3:20 PM)
- 학부모께 태블릿 화면 보여주며 설명
