# 사이수학 입학테스트 - 화면별 상세 정의서

## 📱 학생용 화면

### 1. 테스트 시작 화면 (Welcome Screen)

**목적**: 학생이 테스트의 목적을 이해하고 편안하게 시작할 수 있도록 안내

**사용자 행동**: 
- 화면을 읽고 테스트가 무엇인지 파악
- '시작하기' 버튼 클릭하여 다음 단계로 이동

**주요 구성요소**:
- 제목: "사이수학 입학테스트"
- 안내 문구: 
  - "총 25문제가 출제됩니다"
  - "각 문제당 4점씩 배점됩니다"
  - "시험 시간 제한은 없습니다"
- [시작하기] 버튼 (큰 사이즈, 눈에 띄는 색상)
- 로고 이미지 (상단)

**다음 단계**: 정보 입력 화면으로 이동

**예외처리**: 없음

---

### 2. 정보 입력 화면 (Student Info Form)

**목적**: 학생의 기본 정보를 수집하고 적절한 시험지를 제공하기 위한 데이터 확보

**사용자 행동**: 
- 이름, 전화번호 입력
- 학년, 학기 선택
- '시작하기' 버튼 클릭

**주요 구성요소**:
- **이름 입력란**
  - Label: "이름"
  - Placeholder: "홍길동"
  - 필수 입력 (*)
  
- **전화번호 입력란**
  - Label: "전화번호"
  - Placeholder: "010-0000-0000"
  - 자동 하이픈 입력
  - 필수 입력 (*)
  
- **학년 선택 (Dropdown)**
  - Label: "학년"
  - 옵션: 초등 1학년 ~ 초등 6학년
  - 필수 선택 (*)
  
- **학기 선택 (Radio Button)**
  - Label: "학기"
  - 옵션: ⚪ 1학기  ⚪ 2학기
  - 필수 선택 (*)
  
- [시작하기] 버튼

**다음 단계**: 
- 성공 시 → 문제 풀이 화면 (1번 문제)
- 실패 시 → 에러 메시지 표시 후 현재 화면 유지

**예외처리**:
- 이름 미입력: "이름을 입력해주세요" (빨간색 텍스트, 입력란 테두리 빨간색)
- 전화번호 미입력: "전화번호를 입력해주세요"
- 전화번호 형식 오류: "올바른 전화번호 형식이 아닙니다 (010-0000-0000)"
- 학년 미선택: "학년을 선택해주세요"
- 학기 미선택: "학기를 선택해주세요"
- 해당 시험지 없음: "해당 학년/학기의 시험지가 없습니다. 관리자에게 문의해주세요." (팝업 or 알림)

### 3. 문제 풀이 화면 (Question View)

**목적**: 학생이 한 문제씩 집중해서 풀 수 있도록 최적화된 환경 제공

**사용자 행동**: 
- 문제 읽기 (이미지 확인)
- 4개 선지 중 1개 선택
- '다음' 버튼 클릭하여 다음 문제로 이동

**주요 구성요소**:
- **상단 진행 바**
  - 현재 문제 번호 표시: "1 / 25"
  - 프로그레스 바 (시각적 진행률, 예: ████░░░░░░ 4%)
  
- **문제 영역**
  - 문제 번호: "문제 1"
  - 문제 이미지 (있는 경우)
    - 중앙 정렬
    - 모바일에서도 잘 보이는 크기
    - 확대 기능 (선택사항)
  
- **선지 영역** (4개 카드 형태)
  - ① [선지 1 내용] 
  - ② [선지 2 내용]
  - ③ [선지 3 내용]
  - ④ [선지 4 내용]
  - 클릭 시 배경색 변경 (선택 표시)
  - 한 번에 하나만 선택 가능
  
- **[다음] 버튼**
  - 선지 선택 전: 비활성화 (회색)
  - 선지 선택 후: 활성화 (파란색)
  - 화면 하단 고정

**다음 단계**: 
- 1~24번 문제: 다음 문제로 이동
- 25번 문제: 제출 확인 팝업 → 채점 결과 화면

**예외처리**:
- 선지 미선택 시 버튼 클릭: 버튼 비활성화로 원천 차단
- 브라우저 뒤로가기: "테스트를 종료하시겠습니까? 현재까지 답안은 저장되지 않습니다" 확인 팝업
- 25번 문제 제출 클릭: "제출하시겠습니까? 제출 후에는 수정할 수 없습니다" 확인 팝업

### 4. 채점 결과 화면 (Result Screen)

**목적**: 학생에게 즉각적인 피드백 제공 및 성취감 전달

**사용자 행동**: 
- 점수 확인
- 결과 읽기
- 화면 종료 or 홈으로 이동

**주요 구성요소**:
- **점수 표시 (대형 강조)**
  - 총점: "84점" (큰 폰트, 중앙)
  - 만점 기준: "/ 100점"
  
- **상세 정보**
  - 정답 개수: "✓ 21개 정답"
  - 오답 개수: "✗ 4개 오답"
  - 정답률: "84%"
  
- **메시지 영역**
  - 90점 이상: "🎉 훌륭합니다! 매우 잘했어요!"
  - 70~89점: "👍 잘했습니다! 좋은 결과예요!"
  - 70점 미만: "💪 다음에 다시 도전해보세요!"
  
- **안내 문구**
  - "결과는 관리자가 확인할 수 있습니다"
  - "곧 연락드리겠습니다"
  
- [닫기] 또는 [처음으로] 버튼

**다음 단계**: 
- 닫기: 브라우저 탭 닫기 안내
- 처음으로: 시작 화면으로 이동 (재응시 가능)

**예외처리**: 없음 (채점 완료 후 항상 표시)

## 🖥️ 관리자용 화면

### 5. 관리자 홈 화면 (Admin Dashboard)

**목적**: 관리자의 메인 허브, 주요 기능에 빠르게 접근

**사용자 행동**: 
- 원하는 메뉴 선택
- 최근 통계 확인 (선택사항)

**주요 구성요소**:
- **헤더**
  - 제목: "관리자 페이지"
  - 로그아웃 버튼 (우측 상단)
  
- **메인 메뉴 (카드 형태)**
  - 📝 시험지 만들기
    - 설명: "새로운 시험지 생성"
    - 클릭 → 시험지 생성 화면
  
  - 📊 결과 보기
    - 설명: "학생 응시 결과 조회"
    - 클릭 → 결과 조회 화면
  
- **대시보드 (선택사항)**
  - 오늘 응시 학생 수
  - 전체 시험지 개수
  - 최근 평균 점수

**다음 단계**: 
- 시험지 만들기 → 시험지 생성 화면
- 결과 보기 → 결과 조회 화면

**예외처리**: 
- 로그인 세션 만료: 로그인 화면으로 리다이렉트

---

### 6. 시험지 생성 화면 - 학년/학기 선택

**목적**: 시험지의 대상 학년과 학기 설정

**사용자 행동**: 
- 학년 버튼 선택 (1~6학년)
- 학기 버튼 선택 (1학기/2학기)
- '다음' 버튼 클릭

**주요 구성요소**:
- **학년 선택** (버튼 그리드)
  ```
  [ 초1 ] [ 초2 ] [ 초3 ]
  [ 초4 ] [ 초5 ] [ 초6 ]
  ```
  - 선택 시 버튼 색상 변경
  
- **학기 선택** (토글 버튼)
  ```
  [ 1학기 ] [ 2학기 ]
  ```
  
- [다음] 버튼

**다음 단계**: 문제 입력 화면 (1번 문제)

**예외처리**:
- 학년 미선택: 버튼 비활성화
- 학기 미선택: 버튼 비활성화
- 해당 학년/학기 시험지 이미 존재: "이미 시험지가 존재합니다. 수정하시겠습니까?" 확인 팝업

---

### 7. 시험지 생성 화면 - 문제 입력

**목적**: 25개 문제를 하나씩 입력

**사용자 행동**: 
- 이미지 업로드 (선택)
- 선지 4개 입력
- 정답 선택
- '다음 문제' 또는 '저장' 버튼 클릭

**주요 구성요소**:
- **상단 진행 표시**
  - "문제 1 / 25" (왼쪽)
  - 임시저장 버튼 (오른쪽)
  
- **문제 이미지 영역**
  - [이미지 업로드] 버튼
    - 클릭 → 파일 선택 다이얼로그
    - 지원 형식: JPG, PNG (안내 텍스트)
    - 최대 크기: 5MB (안내 텍스트)
  - 업로드 후: 이미지 미리보기 + [삭제] 버튼
  
- **선지 입력 영역**
  - ① [선지 1 입력란] (텍스트 입력)
  - ② [선지 2 입력란]
  - ③ [선지 3 입력란]
  - ④ [선지 4 입력란]
  - 각 입력란 옆에 글자 수 표시 (예: 0/100자)
  
- **정답 선택 영역**
  - "정답을 선택하세요"
  - ⚪ ①  ⚪ ②  ⚪ ③  ⚪ ④ (라디오 버튼)
  
- **하단 버튼**
  - [이전 문제] (1번 문제에서는 비활성화)
  - [다음 문제] (1~24번)
  - [저장하기] (25번 문제)

**다음 단계**: 
- 다음 문제: 다음 번호 문제 입력 화면
- 저장하기: 저장 확인 팝업 → 성공 메시지 → 관리자 홈

**예외처리**:
- 선지 미입력 (1개라도): "모든 선지를 입력해주세요" (빨간색 텍스트)
- 정답 미선택: "정답을 선택해주세요"
- 이미지 용량 초과: "5MB 이하의 이미지만 업로드 가능합니다"
- 이미지 형식 오류: "JPG 또는 PNG 파일만 가능합니다"
- 저장 실패: "저장에 실패했습니다. 다시 시도해주세요" (에러 팝업)
- 페이지 이탈 시: "작성 중인 내용이 있습니다. 나가시겠습니까?" 확인 팝업

### 8. 결과 조회 홈 화면 (Result List)

**목적**: 응시한 학생들의 목록을 보고 상세 결과로 접근

**사용자 행동**: 
- 학생 목록 스크롤
- 검색 또는 필터 사용
- 특정 학생 선택 (행 클릭)

**주요 구성요소**:
- **검색 및 필터 영역** (상단)
  - 검색창: "이름 또는 전화번호 검색" (Placeholder)
  - 필터 버튼: 
    - 학년별 (전체/초1/초2/.../초6)
    - 학기별 (전체/1학기/2학기)
    - 기간별 (전체/오늘/최근 1주일/최근 1개월)
  
- **학생 목록 테이블**
  | 이름 | 학년 | 학기 | 응시횟수 | 최근응시일 | 최고점수 |
  |------|------|------|----------|-----------|---------|
  | 홍길동 | 초3 | 1학기 | 3회 | 2025.11.10 | 84점 |
  | 김철수 | 초4 | 2학기 | 1회 | 2025.11.09 | 92점 |
  
  - 행 클릭 가능 (hover 효과)
  - 페이지네이션 (한 페이지 20명)
  
- **정렬 기능**
  - 컬럼 클릭 시 오름차순/내림차순 정렬
  - 기본: 최근응시일 내림차순

**다음 단계**: 학생 선택 → 학생 상세 페이지

**예외처리**:
- 검색 결과 없음: "검색 결과가 없습니다" (빈 테이블 표시)
- 데이터 없음: "아직 응시한 학생이 없습니다" (안내 문구)

### 9. 학생 상세 페이지 (Student Detail)

**목적**: 특정 학생의 모든 응시 이력을 날짜별로 확인

**사용자 행동**: 
- 학생 정보 확인
- 응시 이력 목록에서 특정 시험 선택

**주요 구성요소**:
- **학생 정보 카드** (상단)
  - 이름: "홍길동"
  - 전화번호: "010-1234-5678"
  - 총 응시 횟수: "3회"
  - 평균 점수: "78.7점"
  
- **응시 이력 테이블**
  | 응시일 | 학년/학기 | 점수 | 정답수 | 상세보기 |
  |--------|-----------|------|--------|---------|
  | 2025.11.10 14:30 | 초3-1학기 | 84점 | 21개 | [보기] |
  | 2025.10.25 15:20 | 초3-1학기 | 76점 | 19개 | [보기] |
  | 2025.09.15 16:10 | 초2-2학기 | 68점 | 17개 | [보기] |
  
  - 최신순 정렬
  - [보기] 버튼 클릭 → 상세 결과 화면
  
- **통계 차트** (선택사항)
  - 점수 추이 그래프 (꺾은선)

**다음 단계**: [보기] 버튼 클릭 → 상세 결과 화면

**예외처리**: 
- 응시 이력 없음: "응시 이력이 없습니다"

---

### 10. 상세 결과 화면 (Detailed Result)

**목적**: 학생이 푼 25개 문제를 하나하나 확인하고 분석

**사용자 행동**: 
- 문제별 정답/오답 확인
- 스크롤하여 전체 문제 검토
- 인쇄 또는 다운로드

**주요 구성요소**:
- **시험 정보 헤더**
  - 학생 이름
  - 응시 일시: "2025년 11월 10일 14:30"
  - 학년/학기: "초등 3학년 1학기"
  - 총점: "84점 / 100점" (강조)
  - 정답률: "21개 / 25개 (84%)"
  
- **문제별 결과 리스트** (카드 형태)
  각 문제마다:
  ```
  ┌─────────────────────────────────┐
  │ 문제 1                    ✓ 정답 │
  │─────────────────────────────────│
  │ [문제 이미지]                     │
  │                                 │
  │ ① 선지 1                         │
  │ ② 선지 2                         │
  │ ③ 선지 3 ← 학생 답안 (파란색)      │
  │ ④ 선지 4                         │
  │                                 │
  │ 정답: ③번                        │
  └─────────────────────────────────┘
  ```
  
  오답인 경우:
  ```
  ┌─────────────────────────────────┐
  │ 문제 5                    ✗ 오답 │
  │─────────────────────────────────│
  │ [문제 이미지]                     │
  │                                 │
  │ ① 선지 1                         │
  │ ② 선지 2 ← 학생 답안 (빨간색)      │
  │ ③ 선지 3                         │
  │ ④ 선지 4 ← 정답 (초록색)          │
  │                                 │
  │ 정답: ④번                        │
  └─────────────────────────────────┘
  ```
  
- **하단 액션 버튼**
  - [인쇄하기]: 인쇄 친화적 레이아웃 열기
  - [Excel 다운로드]: 결과를 엑셀 파일로 다운로드
  - [목록으로 돌아가기]: 학생 상세 페이지로

**다음 단계**: 
- 인쇄하기: 인쇄 미리보기
- Excel 다운로드: 파일 저장 다이얼로그
- 목록으로: 학생 상세 페이지

**예외처리**:
